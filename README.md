# DirectX11 Lab

This is a personal lab project I created to learn DirectX with AI-assisted guidance.

The goal is to build practical examples step-by-step, with detailed code comments that explain:

- what each part of the code is doing
- why certain DirectX/Win32 choices are made
- what key functions and parameters mean

Because of that, this repo is meant to be both a learning playground and a reference you can come back to.

## Current Focus

- DirectX 11
- Win32 window creation and message loop fundamentals

More examples will be added over time as the lab expands into rendering, resources, shaders, and engine-style structure.

## Why This Exists

I wanted a place to experiment, break things, fix them, and document the process clearly.  
If you are learning DirectX, you can use this repo as a practical companion reference.
